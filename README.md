# Sales Data Analysis using Pandas

This project analyzes sales and customer data using Python. It performs data cleaning, transformation, and visualization using Pandas and Matplotlib.

---

## 📌 Project Overview

The goal of this project is to:
- Analyze sales data
- Clean and preprocess datasets
- Generate insights from customer and sales data
- Visualize trends using charts

---

## 🚀 Features

- Data cleaning and preprocessing
- Handling missing values
- Filtering and sorting data
- GroupBy analysis
- Merging datasets
- Data visualization using Matplotlib

---

## 📂 Files in Project

- `analysis.py` → Main script for data analysis
- `customers.csv` → Customer dataset
